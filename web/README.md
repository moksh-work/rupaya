# Rupaya Web

A modern, responsive web app for Rupaya, built with Next.js. Works on all browsers, laptops, tablets, and mobile devices.

## Features
- Responsive design for all screen sizes
- Modern look and feel
- Fast, accessible, and SEO-friendly

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Run the development server:
   ```sh
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```sh
npm run build
npm start
```

---

This app is ready for deployment to Vercel, AWS Amplify, S3+CloudFront, or any static hosting provider.
